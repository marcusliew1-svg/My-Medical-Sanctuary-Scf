# My Medical Sanctuary / SCF

Repository for My Medical Sanctuary and Stemcell First business materials, website work, documentation, operations, prompts, and technical planning.

## Main folders

- `docs/` - business, legal, operational and planning documents
- `website/` - website and landing page materials
- `content/` - marketing, articles, social media, treatment explanations
- `assets/` - brand assets, image references, logos, design inputs
- `operations/` - SOPs, clinic workflow, team process, checklists
- `prompts/` - reusable ChatGPT/Codex prompts and system instructions
- `vanfo-johor/` - Vanfo Johor ASEAN flagship hub planning materials

## Current purpose

This repository is the working base for building the MMS / SCF digital infrastructure, including future website, documentation, sales materials, and Codex development work.
