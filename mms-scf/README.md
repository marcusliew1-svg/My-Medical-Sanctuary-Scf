# MMS / SCF

Working folder for My Medical Sanctuary and Stemcell First.

## Purpose

This folder keeps the main business, website, clinic, medical wellness, content, operating, and AI/Codex materials for MMS and SCF.

## Subfolders

- `website/` — website planning, pages, copy, and front-end work
- `docs/` — business documents, plans, agreements, internal notes
- `content/` — social media, newsletters, treatment descriptions, articles
- `assets/` — logos, brand assets, images, design references
- `operations/` — clinic operations, SOPs, workflows, staffing, checklists
- `prompts/` — prompts for Codex, ChatGPT, marketing, operations, and automation
